**How to run / view locally**

- Open `index.html` in a browser (double-click or `Open With Live Server`).
- Or serve the folder via a simple HTTP server:

  - Python 3: `python -m http.server 8000` then open `http://localhost:8000`.
  - Node (http-server): `npx http-server -c-1` then open the printed URL.
